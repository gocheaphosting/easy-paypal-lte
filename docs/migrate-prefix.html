Migrate Database Prefix

stores its data in your MySQL database tables. In order to group them together, it uses a prefix, such as ezpp_ in the standalone versions, or wp_ezpp_ in the WordPress plugin version. At times, a system integrator might want to migrate the existing tables to a set of backup tables for safekeeping. This migration tool is perfect for that.

The tool runs pretty much unattended after you enter the new prefix and hit the "Migrate Data" button.
